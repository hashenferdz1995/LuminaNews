# LuminaNews ⚡
### Premium Sports News Blog

A modern, high-intensity sports news blog built with a dark-themed aesthetic, featuring real-time scoreboard mocks, category filtering, and scroll-reveal animations.

## ✨ Features
-   **Dark Mode UI**: Premium charcoal and navy palette with vibrant accents.
-   **Dynamic Hero Section**: Featuring trending sports highlights.
-   **Live Scoreboard Ticker**: Visual indicator for live, upcoming, and finished matches.
-   **Category Filter System**: Responsive pills to filter news (Mocked).
-   **AI-Generated Imagery**: Custom high-resolution sports photography.
-   **Fully Responsive**: Optimised for Desktop, Tablet, and Mobile.

## 🛠️ Tech Stack
-   **Structure**: HTML5
-   **Styling**: Vanilla CSS3 (Custom Design System)
-   **Logic**: Vanilla JavaScript (Animations, Scroll Effects)
-   **Fonts**: Google Fonts (Outfit, Inter)

## 🚀 How to Run
1.  Open `index.html` in any modern web browser.
2.  Alternatively, use a Live Server (like VS Code Live Server) for the best experience.
